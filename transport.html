Стоимость проезда

32 рубля - на трамваях, троллейбусах, автобусах с регулируемым тарифом
16 рублей - для льготников только по транспортной карте (кроме автобусов с нерегулируемым тарифом)
26-32 руб. - на автобусах с нерегулируемым тарифом
По вопросам, возникших при оплате проезда банковской картой (двойная оплата, снятие денег без чека, карта в стоп-листе и т.д.), следует обращаться по телефону 88001003787, электронная почта info@rrtp.ru.
